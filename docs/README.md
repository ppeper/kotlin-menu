## 기능 요구 사항
- [ ] 메뉴 추천을 받을 코치의 이름을 입력 받는다
    - `,`로 구분해서 입력
- [ ] 무작위로 추천할 카테고리 정하기
- [ ] 먹을 메뉴를 추천
- [ ] 각 코치멸 못먹는 메뉴를 입력 받는다
    - 먹지 못하는 메뉴 X -> `빈 값`
    - `,`로 구분해서 입력
## 예외 사항
- [ ] 잘 못된 입력값 `IllegalArgumentException` 발생
- [ ] 한 주에 같은 카테고리 최대 2회
- [ ] 코치는 최대 2~5명 함께 식사 가능
- [ ] 코치의 이름 2~4글자
- [ ] 못 먹는 메뉴 0~2개
- [ ] 코치별 한 주에 중복되지 않은 메뉴 추천
